# Project Canvas

## Nombre del Proyecto

**Análisis del Comportamiento de Clientes en un E-commerce**

## Contexto

Una empresa de comercio electrónico dispone de información histórica sobre pedidos, clientes, productos, pagos, entregas y reseñas.

Aunque existe una gran cantidad de datos disponibles, aún no se comprende completamente cómo se comportan los clientes ni cuáles son los principales factores que impulsan las ventas y la satisfacción.

## Problema

La empresa necesita transformar sus datos históricos en información útil que permita tomar mejores decisiones comerciales y operacionales.

## Objetivo General

Analizar el comportamiento de los clientes mediante técnicas de análisis de datos para identificar oportunidades de mejora en ventas, logística y experiencia del cliente.

## Objetivos Específicos

- Analizar el comportamiento de compra de los clientes.
- Identificar patrones de ventas.
- Evaluar el desempeño de las entregas.
- Analizar la satisfacción de los clientes.
- Detectar oportunidades de mejora para el negocio.

## Stakeholders

- Gerencia Comercial
- Gerencia de Operaciones
- Equipo de Marketing
- Equipo de Logística
- Equipo de Ciencia de Datos

## Datos Disponibles

El proyecto utiliza el conjunto de datos **Brazilian E-commerce Public Dataset by Olist**, el cual incluye información sobre:

- Clientes
- Pedidos
- Productos
- Pagos
- Envíos
- Reseñas
- Vendedores
- Geolocalización

## Indicadores de Éxito

El proyecto se considerará exitoso si permite:

- Identificar patrones relevantes de compra.
- Detectar oportunidades de mejora en la operación.
- Generar indicadores útiles para la toma de decisiones.
- Comunicar los resultados mediante visualizaciones claras.

## Entregables

- Documentación del proyecto.
- Notebook de análisis exploratorio.
- Dashboard de indicadores.
- Informe final con hallazgos y recomendaciones.
